## Description
This repository contains Python programs focused on Object-Oriented Programming (OOP) concepts. It demonstrates how to design and implement programs using classes and objects, helping to build a strong understanding of OOP principles in Python.

## Features
- Creating Classes and Objects
- Constructors (__init__ method)
- Instance Variables and Methods
- Encapsulation
- Inheritance
- Method Overriding
- Basic OOP-based Programs

## Purpose
The purpose of this repository is to understand and practice core OOP concepts in Python through hands-on examples and structured programs.
